# Student-Management-System

A full-stack Student Management System web application that allows administrators to manage student records efficiently. Features include authentication, dashboard statistics, CRUD operations, search functionality, and responsive UI design using HTML, CSS, JavaScript, and MySQL.

---

# Features

✅ Admin Login System  
✅ Dashboard Statistics  
✅ Add Student Records  
✅ Edit Student Details  
✅ Delete Student Records  
✅ Search Students  
✅ Responsive User Interface  
✅ Logout Functionality  

---

# How To Run Project

## Step 1
Clone the repository:

```bash
git clone https://github.com/Naveenjavvadi21/Student-Management-System.git
```

## Step 2
Open the project folder.

## Step 3
Run the project using:
- VS Code Live Server
OR
- XAMPP/WAMP (for PHP backend)

Open in browser:

```bash
http://localhost:5500
```
